# 桌面窗口管理器
对桌面上的窗口进行一些操作。（仅支持Windows）

<center><a href="#下载并安装">下载并安装</a><b> · </b><a href="#选项说明">选项说明</a></center>

## 下载并安装

在[Releases](https://github.com/xiaoshu312/DwmEx/releases)页面下载安装包并完成安装。

## 选项说明

1. 查找窗口：
   - 窗口标题：要被查找到的窗口的标题。
   - 窗口类：要被查找到的窗口的类。
   - 窗口句柄：要被查找到的窗口的句柄。
2. 查找(F)...：根据填写的信息查找窗口。
3. 设置窗口：
   - 窗口标题：被查找到的窗口标题，可以修改。
   - 设置(S)：将被查找到的窗口的标题设置为填写的标题。
   - 窗口句柄：被查找到的窗口句柄，不可修改。
   - 隐藏(H)：隐藏被查找到的窗口。
   - 最小化(M)：最小化被查找到的窗口。
   - 向下还原(R)：向下还原被查找到的窗口。
   - 最大化(A)：最大化被查找到的窗口。
4. 关闭(C)：关闭此对话框。
